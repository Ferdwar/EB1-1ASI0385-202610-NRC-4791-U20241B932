# LP-EB-4791-Felipe Huamanchumo

## Descripción del proyecto

Burger Point es una Landing Page Responsive desarrollada para promocionar un restaurante especializado en hamburguesas artesanales, acompañamientos y bebidas. La página permite a los usuarios conocer el menú, las promociones, la historia del restaurante, la ubicación y contactar con el negocio mediante un formulario.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

---

## Justificación del diseño

### Paleta de colores

Se utilizó una combinación de colores rojo, blanco y gris para transmitir energía, apetito e identidad visual relacionada con restaurantes de comida rápida.

### Tipografía

Se empleó la fuente Arial por ser sencilla, legible y compatible con todos los navegadores.

### Diseño

La Landing Page presenta una estructura simple y organizada para facilitar la navegación del usuario. Se distribuyeron las secciones en el siguiente orden:

- Header con menú de navegación.
- Hero principal.
- Menú de hamburguesas.
- Promociones.
- Sobre nosotros.
- Ubicación y horarios.
- Formulario de contacto.
- Footer.

El sitio también incorpora un menú hamburguesa para dispositivos móviles y un diseño responsive.
